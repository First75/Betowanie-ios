# Betowanie 2026 Mistrzostwa Świata

Prywatna aplikacja do typowania wyników Mistrzostw Świata 2026 ze znajomymi.

## Funkcje

* Typowanie wyników meczów
* Rankingi i tabela punktowa
* Prywatne ligi znajomych
* Śledzenie trafionych typów
* Aktualne wyniki i terminarz
* Prosty i szybki interfejs

## O aplikacji

Aplikacja została stworzona do prywatnego użytku wśród znajomych i fanów piłki nożnej.

Projekt nie jest aplikacją hazardową i nie umożliwia obstawiania prawdziwych pieniędzy.

## Dostęp

Aplikacja dostępna jest prywatnie przez TestFlight.
